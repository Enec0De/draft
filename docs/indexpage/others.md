警告框 { #admonitions }
-----------------------

警告框的常用类型为：`note`，`info`，`tip`，`warning`。

其他可选：`abstract`，`success`，`question`，`failure`，`danger`，`bug`，`example`，`quote`。

``` markdown title="Admonitions"
!!! warning "Warning"

    尽量保证简洁，不乱用警告框。
```

<div class="result" markdown>

!!! warning "Warning"

    尽量保证简洁，不乱用警告框。

</div>
