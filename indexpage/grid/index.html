    网格¶

    只需要使用带有 grid 类的 div 标签包裹一组区块，就可以将任意区块元素排列成网格布局。

    Info

    列表语法是卡片网格的快捷写法。

    对有序或无序的列表，该语法将其包裹于包含 grid 和 cards 两个类的 div 元素内。

    列表语法¶

    有两种写法。

    Note

    两种写法等价，后者多用于组合不同类型的块。

    每个卡片的内容可以不止一行。
